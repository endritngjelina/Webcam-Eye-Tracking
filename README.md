# Webcam-Eye-Tracking

Webcam-Eye-Tracking is a browser-based application for real-time eye tracking using your webcam. Each HTML file in the repository offers a different implementation or experimental interface for capturing and visualizing eye movement, making it useful for demonstrations, prototyping, or educational purposes.

## Features

- Real-time eye tracking via webcam
- Multiple HTML-only app versions for varied experimentation
- No installation or dependencies required
- Showcases simple gaze tracking and visualization
